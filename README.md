# Zed-ziggy

[Ziggy](https://github.com/kristoff-it/ziggy) extension for [Zed](https://zed.dev).

Provides syntax highlighting and outlines for Ziggy documents and Ziggy schemas.

To benefit from the LSP, please install the ziggy executable locally and add it to your path.

Note: Tree-sitter grammars are from [kristoff-it/ziggy](https://github.com/kristoff-it/ziggy) subdirectories:
- [tree-sitter-ziggy](https://github.com/kristoff-it/ziggy/tree/main/tree-sitter-ziggy)
- [tree-sitter-ziggy-schema](https://github.com/kristoff-it/ziggy/tree/main/tree-sitter-ziggy-schema)
